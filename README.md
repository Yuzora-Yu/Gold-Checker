# Prisma Gold Pro - Elite Automated Analysis Dashboard

金（XAU/USD）の価格解析、機関投資家の需給、およびドルの相関を多角的に分析し、AI判定スコアと自動レポートを生成するプロフェッショナル仕様の自動更新型ダッシュボードです。

## 1. データ参照先
本システムは、解析精度とリアルタイム性を両立させるため、以下のソースを組み合わせています。

| 項目 | 参照先 | 取得方法 | 役割 |
| :--- | :--- | :--- | :--- |
| **金価格（解析用）** | Yahoo Finance (`GC=F`) | yfinance (Python) | バックエンドでの指標・需給解析に使用 |
| **ドル指数（解析用）** | Yahoo Finance (`DX=F`, `UUP`) | yfinance (Python) | 金価格との相関係数の算出に使用 |
| **メインチャート** | TradingView (`OANDA:XAUUSD`) | Widget (JS) | スポット価格のリアルタイム表示 |
| **サブチャート** | TradingView (`CAPITALCOM:DXY`) | Widget (JS) | ドル指数の視覚的監視 |

※ 解析には「金先物（GC=F）」を使用しているため、スポット価格（現物）を表示するメインチャートとは数ドルの価格差が発生する場合があります。

## 2. システム構成と仕組み

### A. 自動化エンジン (GitHub Actions)
- **15分おきデータ更新**: 月曜 08:00 〜 土曜 06:00 (JST) の間、ダッシュボードのデータを最新の状態に保ちます。
- **1時間おきX投稿**: 月曜 〜 金曜 08:00 〜 24:00 (JST) の間、解析結果を自動投稿します。
- **手動実行可能**: GitHub上またはダッシュボードのボタンから、任意のタイミングで解析を即時実行できます。手動実行時はXへの投稿は行われません。

### B. 先行指標ロジック (analyze.py)
従来のテクニカル指標に加え、プロフェッショナルな視点を導入しています。
- **CISD (機関投資家の需給ポイント)**: 急騰・急落の起点となった大口注文の溜まり場を特定し、強力なレジサポとして可視化。
- **真空地帯 (ボリューム空白地帯)**: 出来高が薄く、価格が加速（突き抜ける）しやすい領域を特定。
- **時間軸の同期 (5m & 30m)**: 短期と中期のトレンドが一致した「期待値最大化ポイント」を検知。
- **スコアリング**: RSI、乖離率、ATRボラティリティを組み合わせた統合スコア算出。

### C. X（旧Twitter）自動レポート
- **アカウント**: [@FX_Gold_Bit_JP](https://x.com/FX_Gold_Bit_JP)
- **投稿内容**: 最新価格、解析スコア、トレンド同期状態、CISD価格帯を含むレポート。
- **自動チャート生成**: 需給ラインを点線で描画した直近のローソク足チャート画像を自動添付。

### D. フロントエンド (GitHub Pages)
- **同期バッジ**: 時間軸が同期している際に発光エフェクトで通知。
- **発光シグナル**: スコアに基づき、背景が「GOLDEN-GLOW（買い）」や「DEATH-GLOW（売り）」に変化。
- **レポートコピー**: AIへの相談やSNS投稿に最適なフォーマットをワンクリックでコピー可能。

## 3. ファイル構成
- `.github/workflows/main.yml`: 2系統の実行スケジュールと環境変数管理
- `analyze.py`: 需給解析、画像生成、およびX API連携ロジック
- `index.html`: ダッシュボードUIおよびデータ連携スクリプト
- `data.json`: フロントエンドとバックエンドを繋ぐ解析結果ファイル
