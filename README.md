# Prisma Gold Pro - Automated Analysis Dashboard

金（XAU/USD）の価格、テクニカル指標、およびドル指数（DXY）との相関を多角的に分析し、AI判定スコアを出力する自動更新型ダッシュボードです。

## 1. データ参照先
本システムは以下の外部データソースを組み合わせて動作しています。

| 項目 | 参照先 | 取得方法 | 役割 |
| :--- | :--- | :--- | :--- |
| **金価格（解析用）** | Yahoo Finance (`GC=F`) | yfinance (Python) | バックエンドでの指標計算に使用 |
| **ドル指数（解析用）** | Yahoo Finance (`DX=F`, `UUP`) | yfinance (Python) | 金価格との相関係数の算出に使用 |
| **メインチャート** | TradingView (`OANDA:XAUUSD`) | Widget (JS) | リアルタイムチャートの表示 |
| **サブチャート** | TradingView (`CAPITALCOM:DXY`) | Widget (JS) | ドル指数の視覚的監視 |

## 2. システム構成と仕組み
### A. 自動化エンジン (GitHub Actions)
- **スケジュール実行**: 平日の日本時間 9:00 〜 翌 6:00 の間、15分おきに自動起動します。
- **実行環境**: GitHub提供の仮想サーバー（Ubuntu）上で解析スクリプトを実行し、最新データを `data.json` に保存・コミットします。

### B. 解析ロジック (Python / analyze.py)
- **RSI (14)**: 直近14期間の相対力指数から「買われすぎ・売られすぎ」を判定。
- **25日乖離率**: 価格が平均値からどれほど離れているかを算出し、逆張りの優位性を測定。
- **ATRボラティリティ**: 価格変動の勢い（強さ）を検知し、スコアを動的に補正。
- **相関分析**: 金とドル指数の逆相関強度をリアルタイムに数値化。

### C. フロントエンド (GitHub Pages / HTML)
- **データ連携**: ブラウザが `data.json` を1分ごとに読み込み、UIを自動更新します。
- **AI判定アドバイス**: 解析スコアに基づき、「押し目買い」「戻り売り」「静観」の戦略を表示。

## 3. ファイル構成
- `.github/workflows/main.yml`: 実行スケジュールと権限設定
- `analyze.py`: データ取得・解析・判定ロジック
- `index.html`: ユーザーインターフェース
- `data.json`: 中継データファイル
